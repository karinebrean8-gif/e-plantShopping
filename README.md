# 🌿 Paradise Nursery App

The Paradise Nursery App is a simple front-end web application built using React and Redux Toolkit. The main goal of this project is to demonstrate how modern state management works in a real-world e-commerce scenario.

---

## 🚀 Features

- 🌱 Display a list of plants/products
- 🛒 Add items to cart
- ➕ Increase / decrease quantity
- ❌ Remove items from cart
- 💰 Dynamic cart state management using Redux

---

## 🧠 Technologies Used

- React.js
- Redux Toolkit
- JavaScript (ES6)
- CSS
